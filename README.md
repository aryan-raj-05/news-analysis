## Running the App

### Backend

1. Create a virtual environment:

   ```bash
   python -m venv .venv
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file and include:

   ```bash
   GOOGLE_API_KEY=<your_api_key>
   ```

4. Start the backend:

   ```bash
   python app.py
   ```

### Frontend

1. Ensure Node.js and npm are installed.

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the frontend:

   ```bash
   npm run dev
   ```

   or

   ```bash
   npm start
   ```
